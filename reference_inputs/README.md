# Reference inputs

This directory contains only small, publication-specific frozen inputs that are
required to replay the manuscript workflow but are not regenerated by the
ArticleV1 case-study stages themselves.

`transition_policies/article_v1_routing_snapshot.json` is a compact routing
snapshot reconstructed from the persisted ArticleV1 E2 audit records in the
source project. The supplied source ZIP did **not** contain the historical
`posterior_trace.nc` used by the original Bayesian MCMC transition-policy study.
Therefore the default publication reproduction uses this snapshot to reproduce
the 12 representative DEGA executions and their figures. It does not claim to
re-fit the historical PyMC/NUTS posterior.

When the original MCMC posterior artifacts are available under
`Output/Models/TransitionPolicies/BayesianMCMC/`, the canonical runtime can use
them instead.
